# Learning Log 
is a network journal system for tracking the information you </br> 
have received on a specific topic.

The **Learning Log** web application **helps** the user keep a journal </br>
of topics of interest to him and create entries in the journal </br>
while studying each topic. The Learning Log home page contains </br>
a description of the site and invites the user to register or </br>
enter their credentials. After successful login, the user can </br>
create new topics, add new entries, read and edit existing entries.
